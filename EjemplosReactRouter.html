React Router - Ejemplos prácticos

Navegación básica con <Link>

<Link> permite navegar entre rutas sin recargar la página. Es equivalente a una etiqueta <a>, pero manejada por React Router.

<Link to="/">Inicio</Link>
<Link to="/EjemplosSintaxisJSX">SintaxisJSX</Link>
<Link to="/EjemplosEventos">Eventos</Link>

Navegación programática con useNavigate

Permite cambiar de ruta mediante funciones, por ejemplo al hacer clic en un botón.

const navigate = useNavigate();

<button onClick={() => navigate("/")}>Inicio</button>
<button onClick={() => navigate(-1)}>Atrás</button>

Mostrar la ruta actual con useLocation

Devuelve un objeto con información sobre la URL actual, como pathname, search y state.

const location = useLocation();
return <div>Estás en la ruta: {location.pathname}</div>;

Leer parámetros de consulta con useSearchParams

Permite acceder a los ?param=value de la URL.

const [searchParams] = useSearchParams();
const filtro = searchParams.get("filtro");

return <p>Buscar término: {filtro}</p>;

Leer parámetros de ruta con useParams

Utiliza los valores definidos en la ruta como /user/:userId para obtener datos dinámicos de la URL.

const { userId } = useParams();
return <div>Perfil del Usuario con ID: {userId}</div>;

Navegar enviando datos (estado) con navigate(..., { state })

Permite enviar información entre componentes sin usar query params ni rutas dinámicas.

navigate("/NavegarRecibiendoEstado", {
  state: { message: "Hola desde NavegarEnviandoEstado" }
});

Recibir estado desde navegación con useLocation().state

Se accede a los datos enviados mediante navigate(..., { state }) usando useLocation().

const location = useLocation();
const state = location.state || {};

return <div>Dato recibido: {state.message}</div>;
