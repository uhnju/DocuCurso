Ejemplos de Renderizado de Listas en React

Lista simple de cadenas de texto

La forma más básica de renderizar listas es utilizando el método map() sobre un array de strings. Es importante usar la propiedad key para ayudar a React a identificar cada elemento.

Más información: Renderizado de listas en React

const frutas = ["Manzana", "Banana", "Cereza", "Melocotón"];

return (
  <ul>
    {frutas.map((fruta, index) => (
      <li key={index}>{fruta}</li>
    ))}
  </ul>
);

Lista de objetos

Cuando se trabaja con objetos, lo ideal es que cada objeto tenga un id único que se pueda usar como key. Esto mejora el rendimiento y evita advertencias de React.

const usuarios = [
  { id: 1, nombre: "Juan" },
  { id: 2, nombre: "María" },
  { id: 3, nombre: "Carlos" },
];

return (
  <ul>
    {usuarios.map((usuario) => (
      <li key={usuario.id}>{usuario.nombre}</li>
    ))}
  </ul>
);

Lista ordenada al renderizar

Antes de mapear los elementos, puedes ordenarlos usando sort(). Es recomendable clonar el array con slice() para no mutar el original.

const frutas = ["Manzana", "Banana", "Cereza", "Melocotón"];

return (
  <ul>
    {frutas
      .slice()
      .sort()
      .map((fruta, index) => (
        <li key={index}>{fruta}</li>
      ))}
  </ul>
);

Filtrado de lista (números pares)

El método filter() te permite mostrar solo ciertos elementos del array. En este caso, se filtran los números pares antes de renderizarlos.

Esto se puede combinar con map() en una sola cadena.

const numeros = [1, 2, 3, 4, 5, 6, 7, 8, 9];

return (
  <ul>
    {numeros
      .filter((n) => n % 2 === 0)
      .map((n, index) => (
        <li key={index}>{n}</li>
      ))}
  </ul>
);
