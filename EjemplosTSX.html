Ejemplos de TypeScript en Componentes React

1. Tipar props simples

Se define una interfaz para las props que recibe el componente.

interface SaludoProps {
  nombre: string;
  edad: number;
}

const Saludo = ({ nombre, edad }: SaludoProps): JSX.Element => {
  return <p>Hola, {nombre}. Tienes {edad} años.</p>;
};

2. Props opcionales

El símbolo ? indica que una prop es opcional.

interface AlertaProps {
  mensaje?: string;
}

const Alerta = ({ mensaje }: AlertaProps): JSX.Element => {
  return <div>{mensaje ? mensaje : "Sin mensaje."}</div>;
};

3. Tipar children con React.ReactNode

Se usa para representar cualquier contenido JSX que se pase dentro del componente.

interface TarjetaProps {
  children: React.ReactNode;
}

const Tarjeta = ({ children }: TarjetaProps): JSX.Element => {
  return <div className="card">{children}</div>;
};

4. Retorno explícito como JSX.Element

Buena práctica para mayor claridad en funciones que retornan JSX.

const PieDePagina = (): JSX.Element => {
  return <footer>Aquí va el pié</footer>;
};

5. Tipar arrays

Se especifica que elementos es un array de string.

interface ListaProps {
  elementos: string[];
}

const Lista = ({ elementos }: ListaProps): JSX.Element => {
  return (
    <ul>
      {elementos.map((e, i) => <li key={i}>{e}</li>)}
    </ul>
  );
};

6. Tipar eventos

Los eventos tienen tipos específicos como React.MouseEvent.

const Boton = (): JSX.Element => {
  const handleClick = (e: React.MouseEvent<HTMLButtonElement>) => {
    console.log("Click detectado", e);
  };

  return <button onClick={handleClick}>Haz clic</button>;
};

7. Tipar objetos complejos

Se usan varias interfaces anidadas para representar objetos estructurados.

interface Usuario {
  id: number;
  nombre: string;
}

interface PerfilProps {
  usuario: Usuario;
}

const Perfil = ({ usuario }: PerfilProps): JSX.Element => {
  return <p>ID: {usuario.id}, Nombre: {usuario.nombre}</p>;
};

8. Props genéricas

Componentes genéricos permiten aceptar cualquier tipo T.

interface ElementoProps<T> {
  valor: T;
}

const Elemento = <T, >({ valor }: ElementoProps<T>): JSX.Element => {
  return <div>{JSON.stringify(valor)}</div>;
};

9. Tipar referencias con useRef

Especifica el tipo del elemento DOM referenciado.

const EnfocarInput = (): JSX.Element => {
  const inputRef = useRef<HTMLInputElement>(null);
  return <input ref={inputRef} type="text" />;
};

10. Tipar estado con useState

El tipo del estado se indica explícitamente para evitar inferencias incorrectas.

const Contador = (): JSX.Element => {
  const [contador, setContador] = useState<number>(0);
  return <p>Contador: {contador}</p>;
};
