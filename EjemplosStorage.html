Ejemplos de uso de localStorage en React

Guardar datos en localStorage

localStorage.setItem guarda información como texto. Aquí se guarda un nombre simple.

const guardar = () => {
  localStorage.setItem("nombre", "Juan");
  alert("Datos guardados en localStorage");
};

Leer datos desde localStorage

Se accede a un valor con localStorage.getItem. Es necesario verificar si existe antes de usarlo.

const leer = () => {
  const nombre = localStorage.getItem("nombre");
  if (nombre) {
    setNombre(nombre);
  } else {
    alert("No hay datos guardados");
  }
};

Eliminar un ítem de localStorage

Se usa localStorage.removeItem con la clave deseada.

localStorage.removeItem("nombre");

Limpiar todo el localStorage

Con localStorage.clear() se eliminan todas las claves almacenadas.

localStorage.clear();

Sincronizar estado de React con localStorage

Se inicializa el estado con lo que hay en localStorage y se actualiza en cada cambio usando useEffect.

const [nombre, setNombre] = useState(localStorage.getItem("nombre") || "");

useEffect(() => {
  localStorage.setItem("nombre", nombre);
}, [nombre]);

Guardar y leer objetos JSON

Como localStorage solo almacena strings, se usan JSON.stringify y JSON.parse para guardar y leer objetos.

const [usuario, setUsuario] = useState(() => {
  return JSON.parse(localStorage.getItem("usuario") || '{"nombre":"","edad":""}');
});

useEffect(() => {
  localStorage.setItem("usuario", JSON.stringify(usuario));
}, [usuario]);
