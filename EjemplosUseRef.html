Ejemplos de useRef en React

¿Qué es useRef?

useRef es un hook de React que permite:

  • Acceder directamente a elementos del DOM (como document.querySelector).
  • Almacenar valores mutables que no desencadenan renderizados al cambiar.

Documentación oficial: React Docs – useRef

Acceder a un elemento del DOM

Este patrón es útil para hacer foco en un campo, seleccionar texto, o interactuar directamente con elementos HTML desde React.

const inputRef = useRef<HTMLInputElement>(null);

const enfocarInput = () => {
  inputRef.current?.focus();
};

return (
  <div>
    <input type="text" ref={inputRef} placeholder="Escribe algo..." />
    <button onClick={enfocarInput}>Enfocar input</button>
  </div>
);

Almacenar un valor mutable

useRef también se puede usar como una "caja" para guardar valores entre renderizados sin provocar una nueva renderización.

Es útil para contadores internos, métricas o referencias a datos que no necesitan impactar la interfaz.

const valorRef = useRef(0);

const incrementar = () => {
  valorRef.current += 1;
  console.log("Valor mutable:", valorRef.current);
};

return (
  <div>
    <p>Valor mutable: {valorRef.current}</p>
    <button onClick={incrementar}>Incrementar valor mutable</button>
  </div>
);
