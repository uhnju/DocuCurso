Uso de variables y expresiones

En JSX puedes usar cualquier expresión JavaScript encerrándola entre { }. Esto incluye variables, objetos, funciones, interpolación de cadenas, etc.

Es importante que las expresiones devuelvan algo renderizable (número, cadena, componente, etc.).

Más información: Expresiones en JSX

const nombre = "Fernando";
const numero = 42;
const usuario = {
  nombre: "Juan",
  apellido: "Pérez",
};
const getTexto = (texto) => texto.toUpperCase();

<h2>Hola {nombre}!</h2>
<p>El número es: {numero}</p>
<p>Usuario: {JSON.stringify(usuario)}</p>
<p>Nombre completo: {usuario.nombre} {usuario.apellido}</p>
<p>Templates: {`Nombre: ${usuario.nombre} Apellidos: ${usuario.apellido}`}</p>
<p>Texto en mayúsculas: {getTexto("texto")}</p>
<p>NIF Aleatorio: {generarNifAleatorio()}</p>

Estilos en línea

Los estilos en JSX se definen como objetos de JavaScript, donde las propiedades CSS están en camelCase y los valores deben ser cadenas o números.

Más información: Estilos en línea en React

const estiloEnLinea = {
  color: "blue",
  fontSize: "30px",
  fontWeight: "bold",
};

<p style={estiloEnLinea}>Este es un texto con estilos en línea.</p>

Clases CSS en JSX

En lugar de class como en HTML, en JSX debes usar className para aplicar clases CSS, ya que class es una palabra reservada en JavaScript.

Más información: Atributos JSX

<p className="text-danger">
  Este es un texto con clase CSS.
</p>

Fragmentos JSX

Un componente debe devolver un único elemento padre. Si necesitas agrupar varios sin añadir nodos extra al DOM, puedes usar <React.Fragment> o simplemente <>.

Más información: Fragmentos en React

<React.Fragment>
  <h2>Encabezado dentro de un Fragment</h2>
  <p>Esto forma parte del mismo Fragment.</p>
</React.Fragment>

Comentarios en JSX

Para comentar dentro del retorno JSX, se usan llaves {/* */}. Fuera de JSX, los comentarios son como en JavaScript (// o /* */).

{/* Esto es un comentario en JSX */}

Condicional simple con operador ternario

Para mostrar contenido diferente según una condición, puedes usar el operador ternario {condición ? A : B}.

Esto es útil para alternar contenido visible dentro de JSX.

Más información: Renderizado condicional

const usuarioConectado = false;

<p style={{ color: usuarioConectado ? "green" : "red" }}>
  Usuario conectado: {usuarioConectado ? "Sí" : "No"}
</p>

Renderizado condicional con &&

Si solo quieres mostrar algo cuando se cumpla una condición, puedes usar el operador lógico && como atajo para un “if”.

{usuarioConectado && <p>Usuario Conectado!</p>}

  <p>Contenido de otro componente simplificado</p>;
