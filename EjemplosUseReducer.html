Ejemplo de useReducer en React

¿Qué es useReducer?

useReducer es una alternativa a useState para gestionar estados complejos o con múltiples transiciones. Es ideal cuando el estado depende de acciones específicas (como en Redux).

Documentación oficial: React Docs – useReducer

Contador avanzado con useReducer

Este ejemplo define un reducer que maneja tres acciones: incrementar, decrementar y reiniciar.

El estado es un objeto que contiene la propiedad contador. Cada botón envía una acción al reducer a través de dispatch.

// Reducer
const reducerContador = (estado: { contador: number }, accion: { tipo: string }) => {
  switch (accion.tipo) {
    case "incrementar":
      return { contador: estado.contador + 1 };
    case "decrementar":
      return { contador: estado.contador - 1 };
    case "reiniciar":
      return { contador: 0 };
    default:
      return estado;
  }
};

// Componente
const ContadorAvanzado = () => {
  const [estado, dispatch] = useReducer(reducerContador, { contador: 0 });

  return (
    <div>
      <h3>Contador avanzado</h3>
      <p>Contador: {estado.contador}</p>
      <button onClick={() => dispatch({ tipo: "incrementar" })}>+</button>
      <button onClick={() => dispatch({ tipo: "decrementar" })}>-</button>
      <button onClick={() => dispatch({ tipo: "reiniciar" })}>Reiniciar</button>
    </div>
  );
};
