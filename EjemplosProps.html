Ejemplos de Uso de Props en React

Componente simple con props

Los props permiten pasar datos a componentes. Aquí vemos cómo recibir props como nombre, numero y children. Se usan valores por defecto y renderizado condicional para mayor control.

Más información: Pasar props en React

interface SaludoProps {
  nombre?: string;
  numero?: number;
  visible?: boolean;
  children?: React.ReactNode;
}

const Saludo = ({
  nombre = "usuario desconocido",
  numero,
  visible = true,
  children
}: SaludoProps) => {
  if (!visible) return <p>Saludo no visible</p>;

  return (
    <>
      <p>¡Hola, {nombre}!</p>
      {numero || numero === 0 ? <p>Tu número es: {numero}</p> : null}
      <p>{children}</p>
    </>
  );
};

Componente que recibe prop de tipo objeto

Puedes pasar objetos completos como props. Es útil cuando el componente necesita múltiples propiedades agrupadas en un solo objeto.

interface Usuario {
  nombre: string;
  email: string;
}

interface InfoUsuarioProps {
  usuario: Usuario;
}

const InfoUsuario = ({ usuario }: InfoUsuarioProps) => {
  return (
    <p>Usuario: {usuario.nombre}. Email: {usuario.email}</p>
  );
};

Botón reutilizable con estilo por defecto

Un componente de botón que aplica un estilo base y acepta props adicionales. Ideal para reutilizar con diferentes contenidos o eventos.

interface BotonAzulProps {
  children: React.ReactNode;
  style?: React.CSSProperties;
  [key: string]: any;
}

const BotonAzul = ({ children, style = {}, ...props }: BotonAzulProps) => {
  return (
    <button
      style={{ backgroundColor: "#0077FF", color: "#FFFFFF", ...style }}
      {...props}
    >
      {children}
    </button>
  );
};

Uso de componente externo

Este ejemplo muestra cómo usar componentes de una librería externa (como @dit/spad-rdit_library). Estas librerías definen sus propias props, como variant, size y onClick.

<Button
  size="sm"
  variant="danger"
  onClick={() => alert("Evento onClick")}
  style={{ margin: "10px" }}
>
  Button danger
</Button>
