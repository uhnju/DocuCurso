Ejemplos de useState en React

Contador con paso configurable

useState te permite almacenar y actualizar el estado de componentes funcionales. Aquí se usa para manejar un contador y el valor de incremento.

Además, se ilustra cómo controlar inputs numéricos y usar condicionales de estilo.

Documentación: useState (React Docs)

const [contador, setContador] = useState(0);
const [intervalo, setIntervalo] = useState(1);

const incrementar = () => setContador(contador + intervalo);
const decrementar = () => setContador(contador - intervalo);
const inicializar = () => setContador(0);

return (
  <div>
    <p style={{ color: contador > 10 ? "red" : "blue" }}>Contador: {contador}</p>
    <input type="number" value={intervalo} onChange={handleChange} />
    <button onClick={incrementar}>Incrementar</button>
    <button onClick={decrementar}>Decrementar</button>
    <button onClick={inicializar}>Resetear</button>
  </div>
);

Estado booleano y visibilidad

Este patrón es útil para mostrar u ocultar contenido con un botón. El estado booleano cambia con cada clic usando !estado.

const [visible, setVisible] = useState(true);

return (
  <div>
    <button onClick={() => setVisible(!visible)}>
      {visible ? "Ocultar" : "Mostrar"}
    </button>
    {visible && <p>Este contenido es visible pero puede ocultarse</p>}
  </div>
);

Manejo de arrays con useState

Para manejar listas dinámicas, puedes usar arrays en estado. Al agregar elementos, usa el spread operator [...array, nuevo] para mantener la inmutabilidad.

const [elementos, setElementos] = useState<string[]>([]);

const agregarElemento = () => {
  const nuevoElemento = `Elemento ${elementos.length + 1}`;
  setElementos([...elementos, nuevoElemento]);
};

return (
  <div>
    <button onClick={agregarElemento}>Agregar Elemento</button>
    <ul>
      {elementos.map((el, i) => <li key={i}>{el}</li>)}
    </ul>
  </div>
);

Manejo de objetos como estado

Para estados más complejos como formularios, se usa un objeto. Recuerda usar ...spread para mantener el resto de propiedades sin sobrescribirlas.

Este ejemplo muestra cómo manejar múltiples campos y hacer un envío simulado con alert.

const [persona, setPersona] = useState({ nombre: "", edad: 0 });

const actualizarNombre = (e) => setPersona({ ...persona, nombre: e.target.value });
const actualizarEdad = (e) => setPersona({ ...persona, edad: parseInt(e.target.value) });

const handleSubmit = (e) => {
  e.preventDefault();
  alert(JSON.stringify(persona));
};

return (
  <form onSubmit={handleSubmit}>
    <label>
      Nombre: <input value={persona.nombre} onChange={actualizarNombre} />
    </label><br />
    <label>
      Edad: <input type="number" value={persona.edad} onChange={actualizarEdad} />
    </label>
    <button type="submit">Enviar</button>
  </form>
);
